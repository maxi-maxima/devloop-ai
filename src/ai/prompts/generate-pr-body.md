Generate a GitHub pull request body for an automated CI fix.

Include:

## Summary
Briefly explain what changed.

## Root Cause
Explain what caused the CI failure.

## Fix
Explain the patch.

## Validation
List test commands and results.

## Safety Checks
Mention modified file count, forbidden file check, and dry-run status.

## DevLoop Metadata
Include:
- attempt count
- model/provider
- generated by DevLoop AI

Tone:
Clear, professional, transparent.

Do not overclaim.
If confidence is low, say human review is recommended.
